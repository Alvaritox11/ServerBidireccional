Control Omniseekers Robots

How to use

1. Select the camera

You can select the camera you want to see by clicking on the "Select" button. You can select 1, 2 or 3 robots to control at the same time!

2. Talk to the robot

You can talk to the robot by clicking on the microphone button and saying what you want (Example: right.). The instructions will be followed by the selected robot/s.

3. Control the robot

You can control the robot by clicking on the arrow buttons. Be carefull with the robot you are controlling!
